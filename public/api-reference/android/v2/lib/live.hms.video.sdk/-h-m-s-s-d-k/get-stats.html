Starts the collection of statistics from webrtc. Note this is an experimental feature and may cause unpredictable side effects.

This will return a map of track-ids and the corresponding stats. There is also a special id which represents information about the local peer. This key is live.hms.video.connection.degredation.Peer.LOCAL_PEER
