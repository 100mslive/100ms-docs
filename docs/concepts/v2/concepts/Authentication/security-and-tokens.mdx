## Overview

100ms provides two authentication mechanisms. 

- **Server-Side API Authentication** - Authenticates API requests from your server to 100ms using a [Management Token](/concepts/v2/concepts/Authentication/ManagementToken).
- **Client-side authentication** - [Join Room Authentication](/concepts/v2/concepts/Authentication/JoinRoomAuthentication), authenticates your end users when they attempt to join a Room using 100ms SDKs.
