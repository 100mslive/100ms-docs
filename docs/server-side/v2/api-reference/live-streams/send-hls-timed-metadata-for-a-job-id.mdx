Trigger this request to unpause HLS recording .

<PostRequest title="https://api.100ms.live/v2/live-streams/:job_id/unpause-recording" />
<Request id="req-0">

```bash
curl --location --request POST 'https://api.100ms.live/v2/live-streams/<job_id>/unpause-recording' \
--header 'Authorization: Bearer <management_token>' \
--header 'Content-Type: application/json' \
--data-raw '{
    "payload": "<payload>",
    "duration": <duration>
}'
```

</Request>
