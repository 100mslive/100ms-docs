---
title: Stop live stream for a room
nav: 3.162
---

Use this to stop all live streams running in the room. This returns a paginated response for all live streams that were stopped in the room.

<PostRequest title="https://api.100ms.live/v2/live-streams/room/<room_id>/stop" />
<Request id="req-0">

```bash
curl --location --request POST 'https://api.100ms.live/v2/live-streams/room/<room_id>/stop' \
--header 'Authorization: Bearer <management_token>'
```

</Request>
