Analytics Events APIs are used to fetch events with an HTTP request. At the moment we support getting [track events](/server-side/v2/api-reference/analytics-events/track-events).
<EndpointRequest title="https://api.100ms.live/v2/analytics/events"/>

## Response Object

| Attribute | Description                                                                                                                                                                                                                                                                  |
| --------- |------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| `limit`   | Number of events in the results                                                                                                                                                                                                                                              |
| `total`   | Total number of events present                                                                                                                                                                                                                                               |
| `next`    | Used to fetch next set of results, to be specified in request `start` query param. Present only when there are more events present.                                                                                                                                          |
| `events`  | List of events, refer event page for details (eg. [Track event object](/server-side/v2/api-reference/analytics-events/track-events#track-object))                                                                                                                                |
