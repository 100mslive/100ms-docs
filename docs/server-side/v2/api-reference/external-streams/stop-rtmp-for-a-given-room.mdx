nav: 3.19

Trigger this request to stop rtmp streaming and / or recording .

<PostRequest title="https://api.100ms.live/v2/external-streams/room/:room_id/stop" />
<Request id="req-0">

```bash
curl --location --request POST 'https://api.100ms.live/v2/external-streams/room/<room_id>/stop' \
--header 'Authorization: Bearer <management_token>' \
```

</Request>
