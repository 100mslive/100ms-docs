Use this to stop a specific external stream by its unique identifier.

<PostRequest title="https://api.100ms.live/v2/external-streams/<stream_id>/stop" />
<Request id="req-0">

```bash
curl --location --request POST 'https://api.100ms.live/v2/external-streams/<stream_id>/stop' \
--header 'Authorization: Bearer <management_token>' \
```

</Request>
