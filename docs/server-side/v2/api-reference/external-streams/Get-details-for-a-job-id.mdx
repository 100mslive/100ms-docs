Trigger this request to get details for a given RTMP job ID.

<GetRequest title="https://api.100ms.live/v2/external-streams/:job_id" />
<Request id="req-0">

```bash
curl --location --request GET 'https://api.100ms.live/v2/external-streams/<job_id>' \
--header 'Authorization: Bearer <management_token>'
```

</Request>
