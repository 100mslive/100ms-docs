Use this request to get the recording asset object.

<GetRequest title="https://api.100ms.live/v2/recording-assets/<asset_id>" />
<Request id="req-0">

```bash
curl --location --request GET 'https://api.100ms.live/v2/recording-assets/<asset_id>' \
--header 'Authorization: Bearer <management_token>'
```

</Request>
