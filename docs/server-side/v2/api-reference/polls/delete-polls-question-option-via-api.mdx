This endpoint is used to delete polls option. Other options are re-indexed.

<DeleteRequest title="https://api.100ms.live/v2/polls/<poll_id>/questions/<question_id>/options/<option_id>" />

<Request id="req-0">

```bash
curl --location --request DELETE 'https://api.100ms.live/v2/polls/655bffeecd34739a72637305/questions/1/options/1' \
--header 'Authorization: Bearer <management_token>' \
--header 'Content-Type: application/json'
```

</Request>

<ResponseBox id="res-0" status="204 No Content">

</ResponseBox>

## Path parameters

| Name               | Description                                                                                   |
| :----------------- |  :------- |
| poll_id <br/>         | poll id       |
| question_id <br/>        | question_id starting from 1    |
| option_id <br/>     | option_id starting from 1       |
