---
title: Retrieve Room Codes
nav: 4.14
---

Retrieves room codes for all roles in a Room. You can also retrive room codes for a specific role or only enabled room codes using these arguments in the endpoint. 

<GetRequest title="https://api.100ms.live/v2/room-codes/room/<room_id>" />
<Request id="req-comp-0">

```bash
curl --location --request GET 'https://api.100ms.live/v2/rooms' \
--header 'Authorization: Bearer <management_token>' 
```
</Request>
   
<ResponseBox id="resp-0" status="200 OK">
```json
    {
    	limit: 20,
    	data: [
    		{
    			code: "hkj-sdf-fda",
    			room_id: "new-room-1662723668",
    			role: "host",
    			enabled: true,
    			created_at: <timestamp>,
    			updated_at: <timestamp>,
    		},
    		{
    			code: "wiw-giw-bzs",
    			room_id: "new-room-1662723668",
    			role: "guest",
    			enabled: true,
    			created_at: <timestamp>,
    			updated_at: <timestamp>,
    		}
    	],
    	last: ""
    }
    ```
</ResponseBox>    
 
### Arguments 

| Field   | Type     | Description                                                           | Required |
| :----- | :------- | :----------------------------------------------------------------------- | :------- |
| role | `string` | When specified, it retrieves all room codes for the specified role in the room. | No       |
| enabled | `boolean` | Specify enabled : true or false to retrieve relevant room codes | No       |
| limit | `int` |  | No       |
| start | `string` |  | No       |


 ### Errors
 
 | Description | Code |
 | :--- | :--- |
 | Room not found | 404 |
 | Room is disabled | 403 |
 | Authentication Error | 401 |
 | Role not found | 404 |
 | Invalid query parameter | 400 |
