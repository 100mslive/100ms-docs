This endpoint is used to delete a [role](/server-side/v2/api-reference/policy/create-template-via-api#roles-object). Subsequent requests for deleting role which does not exist gives 404.

<DeleteRequest title="https://api.100ms.live/v2/templates/<template_id>/roles/<role_name>" />
<Request id="req-0">

```bash
curl --location --request DELETE 'https://api.100ms.live/v2/templates/<template_id>/roles/<role_name>' \
--header 'Authorization: Bearer <management_token>'
```

</Request>

<ResponseBox id="res-0" status="204 No Content"></ResponseBox>

## Arguments

| Name                           | Type     | Description                                                                                                        | Required |
| :----------------------------- | :------- | :----------------------------------------------------------------------------------------------------------------- | :------- |
| name <br/> — path param        | `string` | Alias of the role you wish to delete.                                                                              | Yes      |
| template_id <br/> — path param | `string` | Unique identifier of the template to which the role is associated. <br/><br/>**Example**: 627ce5b637300e55a72ba7ed | Yes      |
