Retrieves the details of a specific [role](/server-side/v2/api-reference/policy/create-template-via-api#roles-object). 100ms provides options to retrieve details of a role with their name.

<GetRequest title="https://api.100ms.live/v2/templates/<template_id>/roles/<role_name>" />

<Request id="req-0">

```bash
curl --location --request GET 'https://api.100ms.live/v2/templates/<template_id>/roles/<role_name>' \
--header 'Authorization: Bearer <management_token>'
```

</Request>

## Arguments

| Name                           | Type     | Description                                                                                                        | Required |
| :----------------------------- | :------- | :----------------------------------------------------------------------------------------------------------------- | :------- |
| name <br/> — path param        | `string` | Alias of the role you wish to fetch details for.                                                                   | Yes      |
| template_id <br/> — path param | `string` | Unique identifier of the template to which the role is associated. <br/><br/>**Example**: 627ce5b637300e55a72ba7ed | Yes      |
