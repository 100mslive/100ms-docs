This API retrieves details of all the templates in your account.

<GetRequest title="https://api.100ms.live/v2/templates" />

<Request id="req-0">

```bash
curl --location --request GET 'https://api.100ms.live/v2/templates' \
--header 'Authorization: Bearer <management_token>'
```

</Request>
