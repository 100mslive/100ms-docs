Use this API to resume the recording that is paused for a room.

<PostRequest title="https://api.100ms.live/v2/recordings/room/<room_id>/resume" />
<Request id="req-0">

```bash
curl --location --request POST 'https://api.100ms.live/v2/recordings/room/<room_id>/resume' \
--header 'Authorization: Bearer <management_token>' \
```

</Request>
