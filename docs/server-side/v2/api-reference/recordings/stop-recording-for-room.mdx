---
title: Stop recording for room
nav: 3.142
---

Use this to stop all recordings running in the room. This returns a paginated response for all recordings that were stopped in the room.

<PostRequest title="https://api.100ms.live/v2/recordings/room/<room_id>/stop" />
<Request id="req-0">

```bash
curl --location --request POST 'https://api.100ms.live/v2/recordings/room/<room_id>/stop' \
--header 'Authorization: Bearer <management_token>'
```

</Request>
<ResponseBox id="res-0" status="200 OK">
