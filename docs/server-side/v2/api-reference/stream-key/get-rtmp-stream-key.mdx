Use this request to get the RTMP stream key and URL for a specific room.

<GetRequest title="https://api.100ms.live/v2/stream-keys/room/<room_id>" />
<Request id="req-0">

```bash
curl --location --request GET 'https://api.100ms.live/v2/stream-keys/room/<room_id>' \
--header 'Authorization: Bearer <management_token>'
```
</Request>

<ResponseBox id="res-0" status="200 OK">
```json
{
    "data": [
        {
            "id": "652d295286003bb661b5fc07",
            "key": "<stream_key>",
            "room_id": "<room_id>",
            "customer_id": "<customer_id>",
            "app_id": "<app_id>",
            "enabled": true,
            "url": "<rtmp_url>",
            "created_at": "2023-10-16T12:15:14.725Z",
            "updated_at": "2023-10-16T12:15:14.725Z"
        }
    ],
    "last": "652d295286003bb661b5fc07"
}
```
</ResponseBox>

### Arguments

| Field   | Type     | Description |
|:--------|:---------|:------------|
| room_id | `string` | mandatory   |

### Errors

| Description                   | Code |
|:------------------------------|:-----|
| No active key found           | 404  |
| Room not found                | 404  |
| Room is disabled (not active) | 403  |
| Authentication Error          | 401  |
