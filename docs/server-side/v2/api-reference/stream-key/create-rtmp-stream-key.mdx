Use this request to generate an RTMP stream key and RTMP URL for any room, enabling HLS live streaming from an external RTMP broadcaster to the specified room. 

<PostRequest title="https://api.100ms.live/v2/stream-keys/room/<room_id>" />
<Request id="req-0">

```bash
curl --location --request POST 'https://api.100ms.live/v2/stream-keys/room/<room_id>' \
--header 'Authorization: Bearer <management_token>'
```
</Request>

<ResponseBox id="res-0" status="200 OK">
```json
{
    "id": "652d295286003bb661b5fc07",
    "url": "<rtmp_url>",
    "key": "<stream_key>",
    "room_id": "<room_id>",
    "customer_id": "<customer_id>",
    "app_id": "<app_id>",
    "enabled": true,
    "created_at": "2023-10-16T12:15:14.725963917Z",
    "updated_at": "2023-10-16T12:15:14.725963917Z"
}
```
</ResponseBox>

> Note on disabled rooms
>
> This API will not generate stream keys for disabled 100ms rooms. The rooms need to be enabled either through the 100ms Dashboard or using the [REST API](/server-side/v2/api-reference/Rooms/disable-or-enable).
>

### Arguments

| Field   | Type     | Description |
|:--------|:---------|:------------|
| room_id | `string` | mandatory   |

### Errors

| Description                   | Code |
|:------------------------------|:-----|
| Room not found                | 404  |
| Room is disabled (not active) | 403  |
| Authentication Error          | 401  |
| Active key already exists     | 400  |
