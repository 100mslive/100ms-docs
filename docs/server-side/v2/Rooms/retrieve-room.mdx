Retrieves the details of a specific room. 100ms provides options to retrieve details of a room with their ID or name.

## Retrieve a specific room - by ID

### API endpoint

<GetRequest title="https://api.100ms.live/v2/rooms/<room_id>" />

### Arguments

| Name                  | Type     | Description                                                                                                  | Required |
| :-------------------- | :------- | :----------------------------------------------------------------------------------------------------------- | :------- |
| id <br/> — path param | `string` | Unique identifier of the room you wish to fetch details for. <br/><br/>**Example**: 627cda81ab4f3b56a077dc33 | No       |

## Retrieve a specific room - by name

### API endpoint

<GetRequest title="https://api.100ms.live/v2/rooms?name=<room_name>" />

### Arguments

| Name                    | Type     | Description                                                                       | Required |
| :---------------------- | :------- | :-------------------------------------------------------------------------------- | :------- |
| name <br/> — path param | `string` | Alias of the room you wish to fetch details for. <br/><br/>**Example**: test-room | No       |
