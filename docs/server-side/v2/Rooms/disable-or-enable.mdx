Trigger this request to disable a specific room.

## API Endpoint

<DeleteRequest title="https://api.100ms.live/v2/rooms/<room_id>" />

## Arguments

| Name                       | Type      | Description                                                                                                         | Required |
| :------------------------- | :-------- | :------------------------------------------------------------------------------------------------------------------ | :------- |
| id <br/> — path param      | `string`  | Unique identifier of the room you wish to disable. <br/><br/>**Example**: 627cda81ab4f3b56a077dc33                  | Yes      |
| enabled <br/> — path param | `boolean` | Status of the room to be set. <br/>`Allowed values`: true, false. <br/>Should be set to `false` to disable the room | Yes      |
