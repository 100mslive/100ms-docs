Retrieves the details of a specific active room. 100ms provides options to retrieve details of an active room with their ID.

> **Note**: You can also use this API to check if a room has any active sessions or not.

<PostRequest title="https://api.100ms.live/v2/active-rooms" />
<Request id="req-comp-0">

```bash
curl --location --request GET 'https://api.100ms.live/v2/active-rooms/<room_id>' \
--header 'Authorization: Bearer <management_token>'
```
