## Object

| Argument    | Description                                                                                                                                                                                    |
| ----------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| id          | Unique identifier for the room                                                                                                                                                                 |
| name        | Alias for the room                                                                                                                                                                             |
| customer_id | Unique identifier for your account                                                                                                                                                             |
| session     | Object of type `session`. This object contains array of unique identifier of the peers, enabled/disabled status of recording(SFU & browser), enabled/disabled status of streaming(HLS & RTMP). |

### Session object

| Argument   | Description                                                                                                                                                                                                                                                              |
| ---------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| id         | Unique identifier for the session                                                                                                                                                                                                                                        |
| created_at | Timestamp when the session started.                                                                                                                                                                                                                                      |
| peers      | This object contains array of unique identifier of the peers                                                                                                                                                                                                             |
| recording  | Indicates whether recording(SFU & browser) is enabled or not. Refer to [SFU recording guide](/server-side/v2/Destinations/recording) and [Browser recording guide](/server-side/v2/Destinations/rtmp-streaming-and-browser-recording) for more information.              |
| streaming  | Indicates whether streaming(HLS & RTMP) is started or not. Refer to [RTMP streaming guide](/server-side/v2/Destinations/rtmp-streaming-and-browser-recording) and [HLS streaming guide](/server-side/v2/Destinations/live-streaming-via-dashboard) for more information. |
