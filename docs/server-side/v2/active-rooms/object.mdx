The active room object is the existing room object with additional information like the session object (with an array of peer identifiers). This is not a new object.

<EndpointRequest title="https://api.100ms.live/v2/active-rooms" />

## Object

| Argument    | Description                                                                                                                                                                                    |
| ----------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| id          | Unique identifier for the room                                                                                                                                                                 |
| name        | Alias for the room                                                                                                                                                                             |
| customer_id | Unique identifier for your account                                                                                                                                                             |
| session     | Object of type `session`. This object contains array of unique identifier of the peers, enabled/disabled status of recording(SFU & browser), enabled/disabled status of streaming(HLS & RTMP). |

### Session object

| Argument   | Description                                                                                                                                 |
| ---------- | ------------------------------------------------------------------------------------------------------------------------------------------- |
| id         | Unique identifier for the session                                                                                                           |
| created_at | Timestamp when the session started.                                                                                                         |
| peers      | This object contains array of unique identifier of the peers                                                                                |
| recording  | **Note:** We are working on bringing more improvements to this object, we will update more information once the improvements are available. |
| streaming  | **Note:** We are working on bringing more improvements to this object, we will update more information once the improvements are available. |
