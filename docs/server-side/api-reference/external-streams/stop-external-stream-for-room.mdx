---
title: Stop external stream for room
nav: 3.152
---

Use this to stop all external streams running in the room. This returns a paginated response for all external streams that were stopped in the room.

<PostRequest title="https://api.100ms.live/v2/external-streams/room/<room_id>/stop" />
<Request id="req-0">

```bash
curl --location --request POST 'https://api.100ms.live/v2/external-streams/room/<room_id>/stop' \
--header 'Authorization: Bearer <management_token>' \
```

</Request>
