title: Get external stream
nav: 3.153

Use this request to get the external stream object, at any point after it has been created.

This can be used to fetch recording assets that are generated after the stream has completed.

<GetRequest title="https://api.100ms.live/v2/external-streams/<stream_id>" />
<Request id="req-0">

```bash
curl --location --request GET 'https://api.100ms.live/v2/external-streams/<stream_id>' \
--header 'Authorization: Bearer <management_token>'
```

</Request>
