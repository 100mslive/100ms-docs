Use this to stop a specific live stream by its unique identifier.

<PostRequest title="https://api.100ms.live/v2/live-streams/<stream_id>/stop" />
<Request id="req-0">

```bash
curl --location --request POST 'https://api.100ms.live/v2/live-streams/<stream_id>/stop' \
--header 'Authorization: Bearer <management_token>' \
```

</Request>
