Use this API to resume recording that was [paused](./pause-live-stream-recording) for the live stream.

<PostRequest title="https://api.100ms.live/v2/live-streams/<stream_id>/resume-recording" />
<Request id="req-0">

```bash
curl --location --request POST 'https://api.100ms.live/v2/live-streams/<stream_id>/resume-recording' \
--header 'Authorization: Bearer <management_token>' \
```

</Request>
