-   Parameter hasher: the hasher object

| Name   | Description       |

### ==(_:_:)

```swift
public static func == (lhs: HMSSection, rhs: HMSSection) -> Bool
```

resolves comparing of 2 section objects by differentiating based on unique `identifier`

-   Parameters:

#### Parameters

| Name | Description                   |
| ---- | ----------------------------- |
