# HMSUpdateListener

[androidJvm]\
interface [HMSUpdateListener](index) : [IErrorListener](../-i-error-listener/index)

This is the main listener that the calling application should implement. HMS SDK will send all upates through the callbacks in this interface. Pass the app's implementation while calling [HMSSDK.join](../-h-m-s-s-d-k/join) method
