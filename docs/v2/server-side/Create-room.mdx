## Overview

To allow users to join a 100ms video conferencing session inside your app, you first need to create a `room`.

## Create Room On Dashboard

You can create room on [100ms Dashboard](https://dashboard.100ms.live/create-rooms). While creating a room you can specify it's name, roles or enable recording.

<Image width={2168} height={1212} alt="Create a room" src="/docs/v2/create-a-room.png" />

You can add or modify roles on [100ms Dashboard](https://dashboard.100ms.live/roles).
