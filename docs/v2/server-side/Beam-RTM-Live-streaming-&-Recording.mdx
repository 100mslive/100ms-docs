To stream or record your video streaming web apps built on 100ms infra to any platform that supports RTMP input.

---

<PostRequest
    title="Start Streaming"
    desp="This endpoint is used to start live streaming. "
    url="https://prod-in2.100ms.live/api/v2/beam"
/>

<Tabs items={['Request', 'Response']} id="req" />

<div id="req-0">

### Headers

| Authentication <br/> (_REQUIRED_) | `string` | Authentication Bearer Token. This requires server-side token. |
| --------------------------------- | -------- | ------------------------------------------------------------- |

### Body Paramters

| operation <br/> (_REQUIRED_)  | `string` | Should be equal to "start"                                                                 |
| ----------------------------- | -------- | ------------------------------------------------------------------------------------------ |
| room_id <br/> (_REQUIRED_)     | `string` | the room id for which to start the stream                                                 |
| meeting_url <br/> (_REQUIRED_) | `string` | the meeting url to be streamed.                                                            |
| rtmp_url <br/> (_REQUIRED_)   | `string` | the rtmp url to which you want to stream the meeting. Supports rtmp:// & rtmps://          |

</div>

<Response id="req-1" res="200: OK" resText="Beam has started successfully" />

[Click here to see how to generate server side token](/v2/server-side/Generate-server-side-token)
