Whenever the video is rendered on the SurfaceViewRenderer, we can flip the video frame on the horizontal axis i.e creating a mirror frame.

##  Mirror Rendered Video

```kotlin
fun setMirror(isMirrorHorizontal : Boolean)
```

invoke this method with `true`, to flip it in horizontal axis and `false` to restore it in it's original form.


<Tabs id="switch-device" items={['Kotlin', 'Java']} />

<Tab id='switch-device-0'>

```kotlin
surfaceViewRenderer.setMirror(true)
```
</Tab>
<Tab id='switch-device-1'>

```java
surfaceViewRenderer.setMirror(true);
```

</Tab>
