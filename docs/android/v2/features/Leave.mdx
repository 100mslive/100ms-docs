Once you're done with a call and want to exit, call leave on the HMSSDK instance you created to join it.

<Tabs id="leave" items={['Kotlin', 'Java']} />

<Tab id='leave-0'>

```kotlin
hmsSDK.leave()
```
</Tab>

<Tab id='leave-1'>

```java
hmsSDK.leave();
```
</Tab>
