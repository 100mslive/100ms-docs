## Introduction

Blur filter plugin helps in bluring the non-face background of a video frame.
This guide provides an overview of usage of the Blur Filter plugin of 100ms.


## Supported Versions/Resolutions

-   Minimum 100ms SDK version it can work with is `2.9.2`
-   Works best on 15fps
-   Has poor fps on older android phones


## Add dependency

-   Adding the Blur plugin and SDK dependency to your app-level `build.gradle`.

## How to Integrate Blur Filters

### Instantiate 

Instantiate the 100ms Blur Filter plugin like this:

<Tabs id="instanstiate-VirtualBackgroundPlugin" items={['Kotlin']} />

<Tab id='instanstiate-VirtualBackgroundPlugin-0'>

```kotlin
     val hmsSDK = HMSSDK
        .Builder(application)
        .build()

     val blurFilterPlugin  by lazy { HMSBlurFilter(hmsSDK) }

     //call this after onJoin()
     fun addBeautyFilterPlugin() {

        if (hmsSDK.getLocalPeer()?.videoTrack != null) {

            hmsSDK.addPlugin(blurFilterPlugin, object : HMSActionResultListener {
                override fun onError(error: HMSException) {}

                override fun onSuccess() { }

            }, 30)
        }
    }
```

</Tab>


Now let's take a look at the method signature of `HMSBlurFilter`.


### Set Blur Percentage

Adjust blur percentage of a non-person in a video frame. Value is between `0` to `100` default is `75`.

```kotlin
blurFilterPlugin.setBlurPercentage(progress: Int)
```

### Get Blur Percentage

Gets blur percentage of a non-person in a video frame. Value is between `0` to `100` default is `75`.

```kotlin
blurFilterPlugin.getCurrentBlurPercentage()
```
