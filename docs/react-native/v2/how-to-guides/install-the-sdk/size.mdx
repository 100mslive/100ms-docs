The following table lists the size impact on Android for 100ms [Prebuilt](https://www.npmjs.com/package/@100mslive/react-native-room-kit) & [Core SDK](https://www.npmjs.com/package/@100mslive/react-native-hms) packages -


| Increase in Android APK size      | Prebuilt  | Core SDK  |
| --------------------------------- | --------- | --------- |
| (armeabi-v7)                      | **7.4MB** | **7.0MB** |
| (arm64-v8a) **(most common one)** | **8.8MB** | **8.3MB** |
| (x86_64)                          | **9.9MB** | **9.3MB** |
| (x86)                             | **10.2MB**| **9.5MB** |


|                                   | Prebuilt   | Core SDK   |
| --------------------------------- | ---------- | ---------- |
| Increase in JS Bundle size        | **3.23MB** | **0.40MB** |
