# Android - Migration from 0.9.13

This guide is for users migrating from android SDK v0.9.13 to higher versions

---

## Breaking Changes

### Imports

1. The `HMSRTCMediaStream` class imports is changed from `import live.hms.video.webrtc.HMSRTCMediaStream` to `import live.hms.video.HMSRTCMediaStream`
2. Instead of using webrtc classes `AudioTrack`, `VideoTrack` and `MediaStreamuse` `HMSRTCAudioTrack`, `HMSRTCVideoTrack` and `HMSRTCMediaStream` respectively.

#### Change - Import

```java:before
// Before
import live.hms.video.webrtc.HMSRTCMediaStream

import org.webrtc.AudioTrack;
import org.webrtc.MediaStream;
import org.webrtc.VideoTrack;
```

```java:adter
// After
import live.hms.video.HMSRTCMediaStream

import live.hms.video.webrtc.AudioTrack;
import live.hms.video.webrtc.VideoTrack;
```

### GetLocalStreamListener is renamed

-   `HMSClient.GetLocalStreamListener` is renamed to `HMSClient.LocalStreamListener`

#### Change - GetLocalStreamListener
