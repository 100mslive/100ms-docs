# Web - Migrating from 0.9.10

This guide is for users migrating from hmsvideo-web version 0.9.10

---

## Breaking Changes

`stream-remove` event handler signature has changed from

```js
client.on('stream-remove', (room, streamInfo) => {
    // Remove remote stream if needed
});
```

to

```js
client.on('stream-remove', (room, peer, streamInfo) => {
    // Remove remote stream if needed
});
```
