The following table lists the size impact on Android for 100ms [Prebuilt](https://pub.dev/packages/hms_room_kit) & [Core SDK](https://pub.dev/packages/hmssdk_flutter) packages -


| Increase in Android APK size      | Prebuilt  | Core SDK  |
| --------------------------------- | --------- | --------- |
| (armeabi-v7)                      | **5.1MB** | **2.9MB** |
| (arm64-v8a) **(most common one)** | **8.4MB** | **3.6MB** |
| (x86_64)                          | **10.3MB**| **4.2MB** |
| (x86)                             | **10.2MB**| **4.3MB** |


|                                   | Prebuilt   | Core SDK   |
| --------------------------------- | ---------- | ---------- |
| Increase in iOS app size          | **9.60MB** | **8.80MB** |
